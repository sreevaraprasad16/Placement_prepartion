# AI-Based Placement Preparation Dashboard

## Packages to Install (in CMD/PowerShell)
Run these in order after opening CMD in d:/project:

```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

**Packages from requirements.txt**:
- Flask==3.0.3
- Flask-SQLAlchemy==3.1.1
- Flask-Login==0.6.3
- scikit-learn==1.5.0
- pandas==2.2.2
- numpy==2.0.0
- Werkzeug==3.0.3

Then:
```
python app.py
```

Open http://127.0.0.1:5000

Demo: admin/admin

## Features
1. User Authentication Model: Register/Login/Logout
2. Student Profile Model: CGPA/skills/branch/scores
3. Skill Assessment Model: Tests
4. ML Prediction Model: Placement chance
5. Recommendation Model: Tips
6. Chatbot Model: Queries
7. Dashboard Model: Graphs
8. Admin Model: Manage data

Modern responsive design. No errors.

DB/ML auto-setup.
